  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with the HL7 client in minutes
  </p>

  <Steps>
    <Step title="Build the application">
      Clone the repository and build the JAR file using Maven:
      
      ```bash
      git clone https://github.com/deltalogan/hl7-connectivity-providers-client.git
      cd hl7-connectivity-providers-client
      mvn clean package
      ```
      
      The build creates a shaded JAR with all dependencies at `target/connectivity-providers-client-1.0-SNAPSHOT.jar`
    </Step>

    <Step title="Launch the client">
      Run the application with Java 8 or higher:
      
      ```bash
      java -jar target/connectivity-providers-client-1.0-SNAPSHOT.jar
      ```
      
      The login screen will appear where you can select your environment and enter credentials.
    </Step>

    <Step title="Authenticate">
      Enter your credentials in the login form:
      
      - **Email**: Your provider account email
      - **Password**: Your account password
      - **API Key**: Your organization's API key
      - **Environment**: Select DEV, STAGING, or PRODUCTION
      
      <Info>The client automatically manages session tokens and refreshes authentication before expiration.</Info>
    </Step>

    <Step title="Process HL7 transactions">
      Once authenticated, use the main interface to:
      
      - **Check eligibility**: Verify patient coverage and benefits
      - **Register services**: Submit service claims with detailed items
      - **Cancel registrations**: Cancel previously submitted claims
      
      All operations return structured HL7 responses with validation results.
    </Step>
  </Steps>
</div>
