---
title: "Request Models"
description: "Data transfer objects for API requests in the HL7 Connectivity Providers Client"
---

Request models define the structure of data sent to the HL7 connectivity providers. These DTOs are used for authentication and HL7 operations.

## Authentication Requests

### LoginRequest

Authentication request containing user credentials and device information.

**Package:** `com.hl7client.model.dto.request.auth`

<ParamField path="apiKey" type="String" required>
  API key for authentication with the HL7 connectivity provider
</ParamField>

<ParamField path="email" type="String" required>
  User's email address
</ParamField>

<ParamField path="password" type="String" required>
  User's password
</ParamField>

<ParamField path="device" type="DeviceRequest" required>
  Device information for the authentication request
</ParamField>

#### Example

```java
DeviceRequest device = new DeviceRequest(
    "messaging-id-123",
    "device-id-456",
    "My Desktop"
);

LoginRequest request = new LoginRequest(
    "your-api-key",
    "user@example.com",
    "password123",
    device
);
```

### DeviceRequest

Device information included in authentication requests.

**Package:** `com.hl7client.model.dto.request.auth`

<ParamField path="messagingid" type="String" required>
  Messaging identifier for push notifications
</ParamField>

<ParamField path="deviceid" type="String" required>
  Unique device identifier
</ParamField>

<ParamField path="devicename" type="String" required>
  Human-readable device name
</ParamField>

<ParamField path="bloqueado" type="boolean">
  Whether the device is blocked (always `false` in constructor)
</ParamField>

<ParamField path="recordar" type="boolean">
  Whether to remember the device (always `false` in constructor)
</ParamField>

## Enumerations

### Manual

Indicates the processing mode for HL7 operations.

**Package:** `com.hl7client.model.dto.request.hl7`

**Values:**

<ResponseField name="MANUAL" type="String">
  Manual processing mode (value: "0")
</ResponseField>

<ResponseField name="CAPITADOR" type="String">
  Capitation mode (value: "C")
</ResponseField>

<ResponseField name="COMSULTA" type="String">
  Consultation mode (value: "L")
</ResponseField>

<Note>
  The enum values are serialized to their string representations using `@JsonValue`.
</Note>

#### Example

```java
ElegibilidadRequest request = new ElegibilidadRequest();
request.setManual(Manual.MANUAL);
// Serializes as: {"manual": "0"}
```
